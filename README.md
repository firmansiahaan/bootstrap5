# bootstrap5
CSS Framework: BOOTSTARP 5

**Bootstrap 5 is the latest version of the popular front-end framework, offering a modern, responsive, mobile-first design system without jQuery dependency. It’s free, open-source, and widely used for building fast, clean, and adaptive websites.**  

---

## 🚀 Key Highlights of Bootstrap 5
- **No jQuery** → Uses vanilla JavaScript for components, making it lighter and faster.  
- **Responsive Grid System** → Enhanced flexbox-based grid with new utilities.  
- **Improved Forms** → Redesigned form controls with better consistency and accessibility.  
- **Utility API** → Customize and extend Bootstrap with utility classes.  
- **Browser Support** → Works with all modern browsers, but **drops support for Internet Explorer 11**.  
- **Icons** → Bootstrap Icons available as a separate library.  

---

## 📦 How to Get Started
### 1. Include via CDN (Quick Setup)
Add these lines to your HTML file:

```html
<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JS Bundle (includes Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/js/bootstrap.bundle.min.js"></script>
```

### 2. Install via Package Manager
- **npm**: `npm install bootstrap@5`  
- **yarn**: `yarn add bootstrap@5`

## 📊 Comparison: Bootstrap 5 vs. Bootstrap 4

| Feature                | Bootstrap 4            | Bootstrap 5             |
|-------------------------|------------------------|--------------------------|
| **JavaScript**          | Relies on jQuery       | Vanilla JS (no jQuery)   |
| **Grid System**         | Flexbox-based          | Enhanced with utilities  |
| **Forms**               | Basic styling          | Redesigned, more modern  |
| **Browser Support**     | Includes IE11          | No IE11 support          |
| **Icons**               | External libraries     | Bootstrap Icons library  |
| **File Size**           | Larger                 | Smaller, optimized       |

## ⚠️ Things to Keep in Mind
- **Migration from Bootstrap 4**: Some classes have changed (e.g., `.form-group` replaced with utilities).  
- **Performance**: Removing jQuery reduces dependency but may require rewriting old scripts.  
- **Customization**: Use Sass variables and the utility API for tailored designs.
