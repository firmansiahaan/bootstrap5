Class Description
.d-none d-none - Set the position of element to static.
.d-inline d-inline -Set the position of element to relative.
.d-inline-block d-inline-block - Set the position of element to absolute.
.d-inline-block d-inline-block - Force an element to generate a block box that will be flowed with surrounding content i.e. in the same line as adjacent content like inline box.
.d-block d-block - Force an element to generate a block-level box.
.d-table d-table - Force an element to behave like a <table> element.
.d-table-row d-table-row - Force an element to behave like a <tr> element.
.d-table-cell d-table-cell - Force an element to behave like a <td> element.
.d-flex d-flex - Force an element to generate a block-level flex container box.
.d-inline-flex d-inline-flex - Force an element to generate an inline-level flex container box.
